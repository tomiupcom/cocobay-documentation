Customizer

General Settings

Change Logos / Site icon

Customizing > General > Site Identity

  • You can remove/change logo and retina logo.

Change Background

Customizing > General > Background

  • You can change background image

Change theme Colors

Customizing > General > Colors

  • You can pick a color for theme.

Change theme Typography

Customizing > General > Typography

  • You can change typography for body font and title (heading tags: h1,...h6)

Extra Features

Customizing > General > Extra Features

  • On/Off smooth scrolling

Header

Topbar

Customizing > Header > Topbar

  • On/Off topbar
  • Change the content of topbar left.

Header CTA (Call To Action)

Customizing > Header > Header CTA

  • You can change CTA Text
  • You can change CTA URL

Page Settings

Customizing > Page

  • You can change layout of page.

Blog Settings

Customizing > Page

  • You can change layout of blog single and archive.

Footer Settings

Customizing > Footer

  • On/Off Back to top button
  • Change copyright content

Menus

Customizing > Menus

Add menu for all locations

  • Primary menu
  • Footer menu
  • Topbar socials
